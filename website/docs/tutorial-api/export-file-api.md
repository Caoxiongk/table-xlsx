# exportFile
导出文件的方法

参数 | 说明 | 类型 | 默认值
---|---|---|---
columns | 表格列 | array | []
dataSource | 表格数据 | array | []
fileName | 文件名 | string | 'table.xlsx'
hideHeader | 是否显示表头 | bool | false
raw | 是否格式化值的类型 | bool | false
sheetNames | sheet页 | array | ['sheet1']
